# GNN Logistics Landing Page

## Project Overview

This project is a responsive landing page developed for GNN Logistics Inc. as part of the E-Cell Tech Domain Task.

The website provides information about the company and its services while allowing users to submit inquiries through a contact form integrated with Google Sheets.

## Features

### Core Features

* Responsive Navigation Bar
* Call-to-Action Button
* Contact / Inquiry Form
* Form Validation using JavaScript
* Google Sheets Automation
* Social Media Links
* Company Information Section

### Additional Features

* Hero Section
* Services Showcase
* Testimonials Section
* Footer Section
* Responsive Design for Desktop, Tablet and Mobile Devices

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Apps Script
* Google Sheets

## Project Structure

* index.html
* style.css
* script.js
* README.md
* AI_PROMPTS.md

## Setup Instructions

1. Clone the repository.
2. Open the project folder in VS Code.
3. Open index.html using Live Server or any browser.
