# GNN Logistics Landing Page

## Project Overview

This project is a responsive landing page developed for GNN Logistics Inc. as part of the E-Cell Tech Domain Task.

The website showcases the company's services, mission, testimonials, and contact information while providing a modern and user-friendly experience.

## Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action
* Company Overview Section
* Services Showcase
* Testimonials Section
* Contact Form
* Responsive Design for Mobile, Tablet, and Desktop

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* index.html
* style.css
* script.js
* README.md
* AI_PROMPTS.md

## How to Run

1. Download the project files.
2. Open the project folder in VS Code.
3. Open index.html using Live Server or any web browser.
